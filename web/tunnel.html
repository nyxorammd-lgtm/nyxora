How It Works

After installing NYXORA, the system automatically:
1. Pings all 14 transports to measure latency
2. Selects the best transport with lowest latency
3. Creates fixed tunnel between two servers
4. Saves config — connection stays permanent

If you select manually, the tunnel remains fixed between those two servers.

📋 How to Use Configs with NYXORA Tunnels

1️⃣ Create Tunnel

Install NYXORA on both servers. Run:
nyxora connect SERVER_IP

2️⃣ Get Tunnel IP

After tunnel is up, check:
nyxora status
Note the tunnel IP (e.g., 10.100.0.1)

3️⃣ Install Panel

Install Marzban/3X-UI on remote server via scripts above

4️⃣ Configure Panel

In panel settings, set:
• Listen IP: 10.100.0.1 (tunnel IP)
• Port: 443 or custom

5️⃣ Create User

In panel, create user:
• Set traffic limit
• Set expiry date
• Get subscription link

6️⃣ Share Config

Share subscription link or QR code.
Users connect via V2RayNG, SingBox, Clash
